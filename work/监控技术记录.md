#### 设备状态返回值

类似下面：

```
{"errcode": 200, "pid_list": ["2062", "2305"], "pid_num": 2}
```
其中，一定存在的字段是errcode，如果不是200，则是一些未知错误的，会携带错误信息，例如：

```
{"errcode": 400, "errmsg": "ssh login error"}
```
